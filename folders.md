## Folder Structure for Projects

```
[project name]
-- advertisements
-- backup
-- documents
-- seo
-- website
-- website_archive
---- [date of backup (and backup number if more than one)]
---- 02_09_13_01
-- website_content
-- website_design
```

Example:
```
andrewcmaxwell.com
-- advertisements
-- backup
-- documents
-- seo
-- website
-- website_archive
---- 02_09_13_01
-- website_content
-- website_design
